# Aysel-Birthday — сайт-подарок Айсель на 25-летие

Шуточная одностраничка-подарок подруге Ули. Три сцены с убегающими кнопками + финал с сертификатом на йогу с щенками (PuppyClub).

## Контекст
- Имя именинницы: **Айсель**
- ДР: **25-летие, юбилей**
- Шутка контекста: «мы теперь ровесницы» (Уля и Айсель)
- Аналог по механике: `Personal/Lyosha-Date/` (тот же убегающий-нет паттерн)

## Сцены

### 1. «Это ты самая красивая девушка в Питере?»
Да 💖 (стабильна, переход дальше) | Нет (убегает)
Реплики на промах: «нет — это правда», «ну Айсель», «не отнекивайся», «бесполезно отрицать», «ты — да»

### 2. «Это ты ломаешь ноги в танце?»
Я 💃 | Нет (убегает)
Реплики: «весь зал помнит», «видела как ты двигаешься», «ноги отдельно — Айсель отдельно», «не скромничай»

### 3. «Это ты самая прогрессивная зумерка?»
Факт 🌈 | Нет (убегает)
Реплики: «25 — это всё ещё зумер», «ты задаёшь моду», «факт», «не спорь со старшими 😏»

### Финал
- «Оставайся такой чудесной» + «спасибо что даришь свое творчество миру» + «с юбилеем! мы теперь ровесницы»
- Кнопка «открой подарок 🎁» → модалка с сертификатом PuppyClub (2 стороны JPG + кнопка скачать PDF)
- Конфетти + счётчик «ты сказала нет N раз»

## Визуал
- Фон — слайд-шоу из 3 её картин (`assets/painting1-3.jpg`), смена каждые 7 сек, scale-zoom 18s
- Поверх — тёмная вуаль + неоновые блики (фуксия / лайм / циан)
- Палитра: фуксия `#ff2e9a` (акцент), лайм `#c8ff3a` (kicker), циан `#2ad9ff` (script/нет), охра `#ffb938` (подарок), кремовый текст
- Шрифты: Fraunces (display), Caveat (рукопись), Manrope (UI) — те же что у Лёшиного, но контраст инвертирован (тёмный фон, светлый текст)
- Штамп «AYSEL · 25 · JUBILEE» в правом нижнем углу

## Логика убегания
- Та же что у Lyosha-Date: mousemove < 110px → телепорт; touch → автопрыжки 1.3с + телепорт по тапу; после 5 промахов кнопка становится `.is-tiny`

## Структура
```
Aysel-Birthday/
  index.html
  style.css
  script.js
  assets/
    painting1.jpg / painting2.jpg / painting3.jpg  (картины Айсель — из её Downloads/IMG_9088-9090.HEIC)
    certificate-1.jpg / certificate-2.jpg          (две стороны сертификата PuppyClub)
    certificate.pdf                                 (оригинал — для скачивания)
```

## Технологии
- Чистый HTML + CSS + vanilla JS
- canvas-confetti (CDN) — финальный салют
- Без сборки, без npm

## Хостинг
Локально — `open -a "Google Chrome" index.html`. Если выкладывать — GitHub Pages, репо `aysel-birthday`.

## Запреты
- Не превращать в коммерческий лендинг.
- Не убирать «свое» / «нет ё» в финале — это голос Ули.
- Не использовать Safari / Firefox.
